# 微观经济学试题库

福州大学微观经济学课程练习系统。

## 在线访问

**[quiz.codingis.click](https://quiz.codingis.click)**

## 现有题库

| 模块 | 题型 | 题量 |
|------|------|------|
| 微观经济学02 | 选择题 | 31题 |
| 微观经济学03 | 判断题 | 18题 |
| 微观经济学04 | 选择/判断 | 50题 |

## 技术栈

- React 18
- MathJax 3 (LaTeX公式支持)
- 纯CSS，无外部框架
