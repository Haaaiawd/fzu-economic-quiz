## 特性

- **LaTeX 公式支持**：使用 MathJax 3 渲染数学公式，如 $MU_x/P_x=MU_y/P_y$
- **全键盘操作**：支持数字键 1-9 选择、方向键切换、回车键确认/下一题，极速刷题体验
- **进度可视化**：首页列表直观展示每个模块的答题百分比，方便追踪学习进度
- **不定项选择**：支持多选题，需确认答案
- **答题进度保存**：意外关闭页面可继续作答
- **详细解析**：每道题均附生动易懂的解析

## 快捷键说明

| 按键 | 功能 |
|------|------|
| **1-9** | 选择对应序号的选项 |
| **← / →** | 切换上一题 / 下一题 |
| **Enter** | 确认答案 / 查看解析 / 下一题 |

## 技术栈

- React 18
- MathJax 3 (LaTeX 公式渲染)
- 纯 CSS，无外部 UI 框架
